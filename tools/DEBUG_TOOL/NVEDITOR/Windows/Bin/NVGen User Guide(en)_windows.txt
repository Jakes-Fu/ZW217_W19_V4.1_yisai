NVEditor supports command line  7024
Parameters as belowㄩ
NVGen.exe <xxx.prj> <-L> [ [ [ [ [yyy.nv ] dsp.bin ] zzz.txt ] 每W ][-T][每id FileID FilePath][[-m MergeFilePath yyy_config.h]|[-c MergeFilePath Moderm.bin]][-v][-f FixBinFile][-h HashValue.h] [-FastMode]

xxx.prj		[input] it is project file(*.prj). it is must, and others is optional.
-L     		Must use this flag, the NV bin generate is little endian, otherwise is big endian
yyy.nv		[input] it is (*.nv) file, and used to modify the NV item value.
Dsp.bin  	[input] it is dsp bin file, and NVEditor will add it to NV bin file.
zzz.txt 	[output] it is the log file.
-id     	if use this flag, it will requried two other parameters: FileID FilePath. And NVEditor will add this file to NV bin file.
FileID   	[input] File Item ID.
FilePath  	[input] File Path., NVEditor will add it to NV bin file.
-m | -c   	these two flags uesd to deal with Delta NV files according to different projects＊ requirement. 
-m    		if use this flag, it must be require one parameter 每 MergeFilePath, and another parmeter--yyy_config.h is an optional parameter
MergeFilePath   [input] it describe the NV files＊ path, 
yyy_config.h    [output] it will determine the final headerfile naem, its default value is ss_nv_config.h in MergeFilePath directory
-c     		if use this flag, it it must be require one parameter 每 MergeFilePath, which describe the NV files＊ path, and another parmeter is an optional parameter, 
MergeFilePath   [input] it describe the NV files＊ path, 
Moderm.bin      [input|output]it will determine the final bin file, its default value is delta_nv.bin in MergeFilePath directory, or parse .nv files to moderm.bin in marked location.
-f     		[input]if use this flag, it must be requre one parameter〞FixNVBin FilePath, which describe the File Path of ※fixNV.bin§, which will merge to current nvproject bin file
-h     		[output]if use this flag, it must be require one parameter〞HashHeader, which records the hash value generated by the whole nv project, and the results--nvitem.bin must include version information.
-T       	[input]if use this flag, the NV bin without the ※FF§ end-Flag; otherwise, without this flag, the NV bin will include the End-Flag ※FF§.
-v     		if use this flag ,it will be generate the nvitem.bin,which must including the version information.
-FastMode   disable check same item name function

e.g.:  
# generate nvitem.bin directly:
NVGen.exe E:\nvitem\sharkl_nv\nvitem\nvitem.prj -L
# Merge .nv files to yy_config.h file, except generate nvitem.bin(./NVFilePath/ including serval nv files):
NVGen.exe E:\nvitem\sharkl_nv\nvitem\nvitem.prj -L -m E:\nvitem\sharkl_nv\nvitemNVFilePath\  E:\nvitem\sharkl_nv\nvitem\NVFilePath\yy_config.h
