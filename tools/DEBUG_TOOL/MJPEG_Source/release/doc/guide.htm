转换队列

所有在转换队列中的文件在转换开始后，会被逐个转换。转换队列列表显示了一些关于文件的简要信息，包括：文件名、文件大小、文件类型、当前状态(参考文件状态)、目标格式设定和文件完整路径。

 

播放器

在界面上的左侧和右侧分别有2个播放器，左侧用来播放转换前的文件，右侧的用来播放转换后的文件。转换后的文件必须在转换完成后才能播放。

“播放”、“暂停”和“停止”按钮用来控制播放。进度条用来显示当前播放的位置，并可以用来跳转到相应位置开始播放（前提是播放的文件支持Seek）。

 

源文件信息

在此处显示的是源文件的详细信息，包括时长，视频和音频的压缩格式，分辨率，采样率等。

 

视频设定

此处是各种和视频相关的设定，包括视频码率、码率控制方式、视频格式、旋转、缩放、裁减、帧率。

 

音频设定

此处是各种和音频相关的设定，包括音频格式、采样率、码率和声道。

    队列文件

    队列文件是一个用来保存转换编码列表和列表中项目的一些细节的文件。Mp4 Express是以M3U文件加上一些扩展信息的形式来保存队列文件的。Mp4 Express能加载文本文件（文件名一行行的被列出）或M3U文件来作为队列文件。

    转换项目状态：

    准备 - 标记为“准备”的项目将被转换编码
    完成 - 被成功转换编码的项目将被标记为“完成”
    跳过 - 已被成功转换编码过一次的项目，或被用户在转换编码过程中选择成将跳过的 项目，将被标记为“跳过”
    错误 - 在转换编码过程中遇到错误的项目将被标记为“错误”

    目标格式设置

    通常用户选择需要的目标音频和/或视频格式，然后转换编换文件。此外也可以为转换队列中的一个或者一组文件指定目标格式。以下是相应的步骤：

    1.选择你需要的项目以指定和其他项目不同的目标格式
    2.进行音频/视频/容器的设置
    3.选择菜单“项目 -> 设置目标格式模式 -> 私有”，或只按Ctrl + F9，你将看到列表中 目标格式纵列的设置。如果要把它恢复回公共设置，选择菜单“项目 -> 设置目标格式 模式 -> 公共”，或只按Ctrl + Shift + F9
    4.不要忘记恢复音频/视频/容器的设置，因为这些设置对剩余文件是公共设置。
